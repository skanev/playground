> Show how to determine the median of a 5-element set using only 6 comparisons.

This is weird. Here's some ruby code, for a change:

```ruby
def median((a, b, c, d, e))
  a, b = b, a if a > b
  c, d = d, c if c > d
  b, d, c = d, b, a if c < a
  a = e
  a, b = b, a if a > b
  b, d, a = d, b, c if c > a
  a < d ? a : d
end
```
